Repositorio para clase INTRODUCCIÓN A INVESTIGACIÓN CIÉNTIFICA.
---------------------------------------------------------------
_Avances publicados periodicamente._


Este proyecto busca simular de manera simple el comportamiento de un automata celular unidimensional a base de la regla que dicte el usuario (256 reglas en total). Para lograr esto, se utilizó Python para abordar simulaciones. Cada archivo representa mejoras, pero todos tienen la misma finalidad.
